# Eveus EV Charger для Home Assistant

[English](README.md) | **Українська**

> Локальне керування та моніторинг зарядних станцій Eveus у Home Assistant:
> заряджання, жива потужність і вартість, оцінка SOC батареї авто, розклади,
> сповіщення безпеки та готові до автоматизацій сутності.

[Документація](https://abovsh.github.io/eveus/uk/) ·
[Останній реліз](https://github.com/ABovsh/eveus/releases/latest) ·
[Обговорення Home Assistant Community](https://community.home-assistant.io/t/eveus-ev-charger-home-assistant-integration-local-only-hacs/1010628) ·
[Повідомити про проблему](https://github.com/ABovsh/eveus/issues)

Eveus EV Charger підключається до зарядної станції напряму через локальну
мережу та її HTTP API. Інтеграція не потребує хмарного облікового запису й
продовжує працювати без доступу до інтернету.

## Можливості

- керування струмом заряджання, зупинкою, адаптивним режимом, OCPP та захистом
  заземлення;
- жива напруга, струм, потужність, температура, стан заземлення та діагностика;
- енергія й вартість сесії, загальні та скидні лічильники, активні тарифи;
- два розклади заряджання, що зберігаються на самій станції;
- оцінка SOC батареї авто, часу завершення, потрібної енергії та вартості до
  цільового SOC;
- сповіщення Home Assistant Repairs про відсутність заземлення, перегрів,
  витік струму, несправності станції та низький заряд резервної батарейки;
- український та англійський інтерфейс, підтримка кількох зарядних станцій.

## Вимоги

| Вимога | Деталі |
| --- | --- |
| Home Assistant | 2025.1 або новіший |
| Зарядна станція | Eveus 16A, 32A або 48A |
| Мережа | Локальний доступ Home Assistant до HTTP API станції |
| Налаштування | IP-адреса або ім'я хоста, ім'я користувача, пароль і модель |

## Встановлення через HACS

[![Додати репозиторій до HACS](https://my.home-assistant.io/badges/hacs_repository.svg)](https://my.home-assistant.io/redirect/hacs_repository/?owner=ABovsh&repository=eveus&category=integration)

1. Відкрийте HACS і додайте `https://github.com/ABovsh/eveus` як користувацький
   репозиторій типу **Integration**.
2. Встановіть **Eveus EV Charger**.
3. Перезапустіть Home Assistant.
4. Відкрийте **Налаштування → Пристрої та служби → Додати інтеграцію** та
   знайдіть **Eveus EV Charger**.

## Налаштування

1. Введіть IP-адресу, ім'я хоста або повну URL-адресу зарядної станції.
2. Введіть ім'я користувача та пароль.
3. Виберіть модель: **16A**, **32A** або **48A**.
4. Виберіть режим:
   - **Базовий** — керування та моніторинг заряджання;
   - **Розширений** — додатково створює параметри й сенсори SOC та ETA.

Пізніше можна змінити режим через **Налаштувати**, а адресу, облікові дані,
модель і кількість фаз — через **Переналаштувати**.

## Безпека

Небезпечні та проблемні стани відображаються в
**Налаштування → Система → Виправлення**. Інтеграція повідомляє про відсутність
заземлення, вимкнений захист заземлення, перегрів, витік струму, несправності
станції, низький заряд резервної батарейки та неправильний час станції.

Не ігноруйте повідомлення про заземлення, перегрів або витік струму. Зупиніть
заряджання та зверніться до кваліфікованого електрика, якщо небезпечний стан
повторюється.
