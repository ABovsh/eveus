Install Eveus through HACS

  1. Open HACS and add https://github.com/ABovsh/eveus as a custom Integration repository.
  2. Install Eveus EV Charger and restart Home Assistant.
  3. Open Settings, Devices & Services, Add Integration, then search for Eveus EV Charger.
  4. Enter the charger address and credentials, then choose your charger model and integration mode.

Supported chargers: Eveus 16A, 32A, and 48A models reachable from Home Assistant over the local network.
